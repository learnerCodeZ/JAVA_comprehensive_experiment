# 智能家居系统 - 使用指南

## 1. 编译

在项目根目录打开终端，执行：

```bash
javac -d out -sourcepath src -cp "lib/*" src\com\smarthome\Main.java
```

## 2. 运行

```bash
java -cp "out;lib/*" com.smarthome.Main
```

## 3. 登录

弹出登录对话框后输入：

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | admin123 | 管理员 |
| user | user123 | 普通用户 |

## 4. 主界面操作

登录后进入主界面，你可以：

- **左侧房间列表** — 点击"客厅/卧室/厨房/书房"筛选设备
- **中央设备卡片** — 点击任意设备卡片，弹出控制对话框，可以开启/关闭设备
- **右侧控制按钮** — "全部开启"/"全部关闭"所有设备
- **底部日志区** — 显示最近 20 条操作日志
- **菜单栏 → 场景(S)** — 回家/离家/睡眠模式
- **菜单栏 → 网络(N) → 网络控制** — 打开网络面板

## 5. 网络功能测试

1. 点击菜单 **网络(N) → 网络控制**
2. 在弹出面板中点击 **启动服务**，状态变为"运行中 (端口 8888)"
3. 在下方"客户端测试"区域输入命令测试：
   - `GET_ALL` — 获取所有设备状态
   - `TURN_ON:d1` — 开启客厅灯
   - `TURN_OFF:d1` — 关闭客厅灯
4. 点击 **发送**，上方日志区显示通信记录，下方显示响应结果

## 6. 数据持久化验证

1. 在主界面开启几个设备
2. 关闭程序（点窗口关闭按钮）
3. 重新运行 `java -cp "out;lib/*" com.smarthome.Main`
4. 再次登录后，设备状态与关闭前一致（已写入 data/devices.json）

## 7. 确认文件变化

操作后可以检查 data 目录：

```bash
cat data/devices.json
cat data/users.json
```

devices.json 会显示设备的详细状态（亮度、温度、音量等），users.json 会显示用户信息。
